## What Data Can You Extract from ImportYeti?

Every result includes:

- Company/supplier name, type, country, address, geo coordinates
- Total shipment count, most recent shipment date, top suppliers/customers
- HS and HTS code breakdowns with shipment counts and weights
- Trading partners with 12-month and total shipment history
- Shipping lanes (origin port → destination port) with costs, weight, TEU
- Top carriers (SCAC codes) with year-over-year trends
- Container types, sizes, and FCL/LCL split
- Recent bills of lading with product descriptions, weights, routes
- Notify parties with contact info (emails, phone numbers)
- Other addresses and associated companies
- Quarterly growth metrics and shipping cost totals
- Geo coordinates (latitude/longitude), city, state, country
- UFLPA compliance flags
- Continent-level import breakdown
- Chart images (imports per country, top relationships, shipments over time)

## Input Parameters

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `query` | string | *required* | Company name, supplier name, or product keyword |
| `searchType` | enum | By Name | Search by company/product name or by address/location |
| `maxResults` | integer | 100 | Maximum results to return (1–1,000) |
| `type` | enum | Any | Filter by entity type: Supplier or Company |
| `mostRecentShipment` | enum | Any Time | Only entities with shipments in the last 6 or 12 months |
| `minShipments` | integer | — | Minimum total shipment count to include a result |

## How to Scrape ImportYeti with Python

```
from apify_client import ApifyClient

client = ApifyClient("your_token")
run = client.actor("zen-studio/importyeti-scraper").call(run_input={
    "query": "porsche",
    "maxResults": 25,
})

for item in client.dataset(run["defaultDatasetId"]).iterate_items():
    print(f"{item['name']} ({item['countryCode']}) — {item['totalShipments']} shipments")
```

## How to Scrape ImportYeti with JavaScript

```
import { ApifyClient } from 'apify-client';

```

## FAQ

**How many results can I extract?**
No hard limit. Set `maxResults: 0` for unlimited.

**Do I need an ImportYeti account?**
No. The scraper extracts publicly available data without any login.

**How fresh is the data?**
Data is scraped in real time from ImportYeti, so you always get the latest available records.

**Can I filter by country?**
Not directly as an input parameter, but you can use the search query (e.g., "germany automotive") or filter results by `countryCode` in your downstream processing.

**What are HS codes and HTS codes?**
HS (Harmonized System) codes are international product classification codes. HTS (Harmonized Tariff Schedule) codes are the US-specific extension used for import duties. Both are included in the detail output.

## Legal Compliance

This Actor extracts publicly available US import/export records. Users must ensure compliance with applicable data protection regulations (GDPR, CCPA) and ImportYeti's terms of service when using extracted data.
